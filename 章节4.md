### 简单if语句
> **语法**  
> &ensp;&ensp;&ensp;&ensp;if、else属于条件分支语句
```
if(条件){
    //语句1             条件成立：执行语句1
}else{
    //语句2             条件不成立：执行语句2
}
```
> **要点**  
> &ensp;&ensp;&ensp;&ensp;程序流程有三：    
> &ensp;&ensp;&ensp;&ensp;&ensp;&ensp;1. 顺序执行  
> &ensp;&ensp;&ensp;&ensp;&ensp;&ensp;2. 选择执行  
> &ensp;&ensp;&ensp;&ensp;&ensp;&ensp;3. 循环执行  
> &ensp;&ensp;&ensp;&ensp;\*条件分支语句用于选择执行
### 简单if语句代码示例  
```
public class IfDemo{
    public static void main(Sring []args){
        int cash=500;
        char currency='$';
        //最简单的if语句
        //当if语句块中只有一条的指令的时候，可以省略大括号.
        //不推荐使用，尽量使用大括号.
        /*if(cash==400){
            System.out.println("一定是在做梦");
            System.out.println("哈哈");
        }*/
        if(cash>=500){
            System.out.println("买别墅");
        }else{
            System.out.println("公寓");
        }
        
        int a=10;
        int b=1;
        int c;
        /*if(a<b){
            c=a;
        }else{
            c=b;
        }
        System.out.println(c);*/
        
        //条件表达式，它是一个简化的if...else语句
        c=(a<b)?a:b;
        System.out.println(c);
        
        if(cash>=500){
            if(currency=='$'){
                System.out.println("买独栋别墅");
            }else{
                System.out.println("买联排别墅");
            }
        }else{
            System.out.println("买公寓");
        }
        
    }
}
```
### 多重if、else语句  
> **语法**  
```
if(条件1){
    语句1;
}
else if(条件2){
    语句2;
}else{
    语句3;
}
```

```

        if(cash>=500){
            System.out.println("买别墅");

            System.out.println("公寓");
        }
```
